# Selenium Automation Exercises

Repositorio con ejercicios de automatización usando **Selenium WebDriver**.
Proyecto orientado a prácticas de **QA Automation** y demostración para CV / GitHub.

##  Sitio bajo prueba
https://the-internet.herokuapp.com

##  Casos automatizados
- Login exitoso
- Login fallido
- Búsqueda y validación de elementos

##  Tecnologías
- Java
- Selenium WebDriver
- JUnit
- Maven
- ChromeDriver
